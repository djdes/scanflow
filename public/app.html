Как получить Chat ID и Bot Token
  1. 1
    В Telegram найдите @BotFather, напишите /newbot, дайте имя — получите токен вида 123456789:ABC….
  2. 2
    Вставьте токен в поле Bot Token выше.
  3. 3
    Найдите свежесозданного бота по username и нажмите Start (или напишите /start).
  4. 4
    Нажмите кнопку «Найти» возле поля Chat ID — он подставится автоматически, в Telegram придёт подтверждение.
  5. 5
    Нажмите «Сохранить», затем «Отправить тестовое сообщение» для проверки.
