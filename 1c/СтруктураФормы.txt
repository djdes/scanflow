ПОШАГОВАЯ ИНСТРУКЦИЯ СОЗДАНИЯ В КОНФИГУРАТОРЕ:
--------------------------------------------------------------------------------

1. Файл → Новый → Внешняя обработка
2. Имя: ЗагрузкаНакладныхJPGExchange
3. Синоним: Загрузка накладных из JPGExchange

4. ДОБАВИТЬ РЕКВИЗИТЫ (правой кнопкой на "Реквизиты" → Добавить):
   - Создать каждый реквизит из таблицы выше

5. СОЗДАТЬ ФОРМУ:
   - Правой кнопкой на "Формы" → Добавить → Форма
   - Тип: Форма обработки
   - Имя: ФормаОбработки
   - Основная форма: Да

6. В КОНСТРУКТОРЕ ФОРМЫ:
   - Перетащить реквизиты на форму (они появятся как поля ввода)
   - Добавить группы для организации элементов
   - Добавить команды и кнопки
   - Добавить поле ПолеЛога (реквизит формы типа Строка)

7. СКОПИРОВАТЬ КОД:
   - Открыть модуль формы (кнопка "Модуль" внизу редактора формы)
   - Вставить код из файла МодульФормы.bsl

8. СКОПИРОВАТЬ КОД ОБЪЕКТА:
   - В дереве обработки дважды кликнуть на корень (или Ctrl+Alt+M)
   - Вставить код из файла МодульОбъекта.bsl

9. СОХРАНИТЬ:
   - Файл → Сохранить как...
   - Тип: Внешние обработки (*.epf)
   - Имя: ЗагрузкаНакладныхJPGExchange.epf
